Début

liste ← [1, 2, 3, 1, 2, 4, 5, 3, 6]

nouvelle_liste ← []

Si élément n'est pas dans nouvelle_liste Alors

Ajouter élément à nouvelle_liste

FinSi

FinPour

FinSi

Afficher "Liste sans doublons :", nouvelle_liste
